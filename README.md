# jimirOS
it doesn't get any better than this. higher half kernel. microkernel.

##
- ps/2 and usb UHCI support
- AHCI and legacy ATA with ext2
- minimal C std lib

## prereq
- make, bash, i686-elf-gcc and i686-elf-binutils
- use qemu-system-i386

## build
- ./clean.sh
- ./build.sh
- ./iso.sh

## qemu run
- ./qemu.sh

- do help
